Class: ScoreSelector

Composites. ScoreSelector

This node runs the left child nodes (Scorers aka Controllers) , getting a score and index then uses the score to select a corresponing (by same index) on the right children (Executors aka Convos) It works in conjunction with the AddScore action. The AddScore look (up the tree) for the first ScoreSelector and update its score there. This allows nesting of ScoreSelectors - actors can addscores for higher contexts. ONLY SCORERS CAN USE sub-ScoreSelectors - those include other Scorers, and AddScore-only (usually) Executors
